# Oneshot RPI

You are tasked with researching, planning and implementing a feature in one shot.

## Steps to follow after receiving the ticket number

1. use /RPI/1_research_codebase with the given ticket number
2. use /RPI/2_create_plan with the given ticket number
3. use /RPI/2.5_review_plan with the given ticket number
4. use /RPI/3_define_test_cases with the given ticket number
5. use /RPI/4_implement_plan with the given ticket number
6. use /RPI/5_validate_implementation with the given ticket number
You should create the artifacts as the commands suggest.
