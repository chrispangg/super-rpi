# Oneshot RPI

You are tasked with researching, planning and implementing a feature in one shot.

## Steps to follow after receiving the ticket number

1. use SlashCommand() to call /RPI/1_research_codebase with the given ticket number
2. use SlashCommand() to call /RPI/2_create_plan with the given ticket number
3. use SlashCommand() to call /RPI/3_implement_plan with the given ticket number
4. use SlashCommand() to call /RPI/4_validate_implementation with the given ticket number
You should create the artifacts as the commands suggest.
